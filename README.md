#  Unit Converter (cm ↔ inch)

---

##  Overview
The **Unit Converter** is a Python-based command-line application that allows users to convert values between centimeters (cm) and inches (inch). It provides a simple and interactive interface for performing accurate unit conversions.

This project is useful for beginners to understand functions, loops, and exception handling in Python.

---

## Objective
- To build a simple unit conversion tool  
- To perform cm to inch and inch to cm conversions  
- To practice Python functions and loops  
- To handle user input errors using exception handling  

---

##  Features
- Convert centimeters to inches  
- Convert inches to centimeters  
- Menu-driven CLI interface  
- Error handling for invalid input  
- Continuous execution until user exits

Choose an option:
- 1 → cm to inch  
- 2 → inch to cm  
- 3 → Exit  

Enter the value and get the converted result instantly.

---

##  How It Works
- Uses two functions:
  - `cm_to_inch(cm)`  
  - `inch_to_cm(inch)`
  Applied conversion formulas:

   1 inch = 2.54 cm
   1 cm = 0.3937 inch
- Takes user input through a loop  
- Converts values based on selected option  
- Handles invalid input using try-except
